# Проект для дрона

```
1. Установить git
2. Установить node.js и npm
2. Открыть терминал в папке, в которую вы хотите сохранить проект, ввести в терминале команду git clone https://github.com/KarenGrigoryan1999/dron-svelte-project.git
3. Перейти в новую папку папку и внутри нее выполнить команду в терминале npm i
4. Выполнить команду npm run dev -- --open и проект автоматически запустится!
```
